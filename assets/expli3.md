# Callback en Dash

Un **callback** est une fonction qui crée une relation dynamique entre les composants de votre application. Concrètement, il permet de mettre à jour automatiquement un élément (Output) lorsqu'un autre élément (Input) change.

Quand un utilisateur interagit avec un composant input (clic sur un bouton, sélection dans une liste, etc.), le callback déclenche une fonction qui traite cette action et met à jour les composants output en conséquence. C'est ce mécanisme qui rend les applications Dash **interactives et réactives**, permettant une communication bidirectionnelle entre l'interface et le code Python.
