# Layout en Dash

Un **layout** en Dash est la structure visuelle de votre application. Il définit comment les composants sont organisés et affichés sur la page. C'est essentiellement le "squelette" de votre interface utilisateur.

## Éléments principaux d'un layout :

- **Conteneurs** : `dcc.html.Div()` pour organiser les éléments en sections
- **Inputs** : boutons, listes déroulantes, champs de texte pour l'interaction utilisateur
- **Outputs** : graphiques, tableaux, textes pour afficher les résultats
- **Styles CSS** : mise en forme et positionnement des éléments
- **Composants Dash Core** : graphiques interactifs, sliders, calendriers
- **Composants HTML** : titres, paragraphes, images, listes

Le layout est défini en Python et utilise une structure hiérarchique pour organiser tous ces éléments.
