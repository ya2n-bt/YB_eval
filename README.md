# Application Dash MECEN - Ventes d'Avocats

Cette application interactive permet de visualiser et de comparer les données de ventes d'avocats aux États-Unis.

## Prérequis :

Ce projet utilise le gestionnaire de paquets `uv` pour gérer ses dépendances automatiquement. 
Si vous ne l'avez pas encore, installez-le en suivant la [documentation officielle](https://docs.astral.sh/uv/) (par exemple via `pip install uv`)

## Lancement de l'application :

Placez-vous à la racine du projet dans votre terminal et exécutez simplement la commande suivante :

```bash
uv run python app.py
```
